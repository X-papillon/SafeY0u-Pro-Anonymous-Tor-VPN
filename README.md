## 🛠 Installation

### Prerequisites

- Windows 8 or later
- .NET Framework 4.8 or higher
- Internet connection

### Setup

1. Download the latest release from the [Releases](https://github.com/yourusername/SafeY0u-Pro/releases) page

2. Extract the ZIP file to any folder

3. Run `SafeY0u-Pro.exe`

4. Select your preferred browser

5. Click **START TOR**

6. Wait for bootstrap to complete (100%)

7. Configure your browser proxy settings:
SOCKS5 Host: 127.0.0.1
Port: 9050

### Firefox Proxy Configuration

1. Settings → Network Settings
2. Manual proxy configuration
3. SOCKS Host: `127.0.0.1` | Port: `9050`
4. Select "SOCKS v5"
5. Check "Proxy DNS when using SOCKS v5"

## ❓ Frequently Asked Questions

**Q: Is SafeY0u Pro completely free?**
A: Yes, it's 100% free and open-source.

**Q: Does it slow down my internet?**
A: Tor routing may reduce speed due to encryption and relay routing.

**Q: Can I use any browser?**
A: Yes, any browser supporting SOCKS5 proxy.

**Q: Is my data logged?**
A: No, the application does not collect any user data.

**Q: Where is tor.exe stored?**
A: In `%AppData%\safeyoutor\tor.exe` (hidden file)

**Q: How often does Tor IP change?**
A: Approximately every 10 minutes or when requested.
